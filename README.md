# leaflet

## Project setup

```
npm install
npm install leaflet
npm install vue-router@next
```

## Folder

- **static**：[Leaflet - `tutorial`](‘https://leafletjs.com/examples.html’) study

## Features

- 台风实时路径实现 ([浙江水利]('http://typhoon.zjwater.gov.cn/default.aspx))
  - 地图图层切换（基础底图&影像图）
  - 台风路径绘制及动态绘制路径效果。
  - 台风图标 Marker 及气泡信息弹窗
- 旅游相册
- 雨水分布
- Vue3
  - provide - inject
  - v-model : modelValue
